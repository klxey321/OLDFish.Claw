# OLDFish.Claw 品牌与视觉说明

## 1. 品牌名称

统一名称：

- `OLDFish.Claw`

不再使用：

- `OpenClaw Control Center`
- 任何旧项目品牌名

## 2. 品牌定位

`OLDFish.Claw` 不是通用后台，而是：

- 总办主脑控制面板
- 四部门节点调度台
- 机械黑客风控制系统

## 3. 视觉方向

整体方向固定为：

- 机械
- 冷峻
- 黑客感
- 暗色高对比
- 控制台读数风格

不使用：

- 白蓝玻璃产品页风格
- 轻盈营销页风格
- 默认系统后台风格

## 4. 颜色方向

建议基准色：

- 背景：哑黑 / 深灰 / 石墨色
- 正常：荧光绿
- 忙碌：电光青
- 阻塞：警示橙
- 离线：冷灰
- 异常：故障红

## 5. 页面气质

页面要给人的感觉是：

- 主脑在看全局
- 节点在汇报状态
- 操作者在控制机器

不是：

- SaaS 官网
- 普通后台模板
- 电商管理台

## 6. 文案口径

推荐用词：

- 总办主脑
- 节点
- 调度
- 摘要
- 阻塞
- 派单
- 执行
- 验收

避免用词：

- 过于通用的“员工协同平台”
- 旧项目里的 OpenClaw 官方表述
